Установка:

1. Загрузить весь архив модуля через админку OpenCart, меню "Установка расширений" ("Extension Installer")
2. Включить модуль на этой странице
3. Включить модуль на странице модулей
4. Изменить настройки


Что делает этот модуль?

Модуль улучшает страницу События (Events) в OpenCart.
Он добавляет следующий функционал:
1. Возможность редактировать события, изменяя все поля в них.
2. Возможность создавать новые события через админку а не через программный код.
3. Фильтровать существующие события по таким полям: код, trigger, action, статус
